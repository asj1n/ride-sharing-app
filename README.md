# Ride Sharing Application
- Frontend for a Ride Sharing Application, a platform matching drivers and passen-
gers by proximity and destination. Business logic implemented following design patterns and Test-Driven-
Development (TDD), validated with JUnit; persistence via serialization.
- Backend in Java Spring Boot using Quadtrees for location-based matching. Frontend in Vaadin with side tabs,
supporting user and vehicle management, ride creation, updates, reviews, and dynamic matching.
